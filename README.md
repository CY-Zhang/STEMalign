# STEMalign
Automatic STEM alignment with Bayesian optimization based on BoTorch, GP model from GPyTorch

Current libraries:

Python 3.7.3
CUDA 10.1
Tensorflow 2.2.0
Keras 2.3.1
botorch 0.4.0
torch 1.7.1 (CPU only)
gpytorch 1.4.2
